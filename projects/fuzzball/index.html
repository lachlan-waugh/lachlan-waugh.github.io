Installation#

  1. git clone git@github.com:lachlan-waugh/fuzzball.git
  2. cd fuzzball
  3. ./fuzzer --install
  4. ./fuzzer --help

 

Usage#

  1. ./fuzzer binary_name input_name
  2. ./fuzzer --test
  3. ./fuzzer --test <directory containing binaries and their sample input>
